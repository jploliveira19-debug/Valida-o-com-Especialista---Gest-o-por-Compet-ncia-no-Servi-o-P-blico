1. Apresentação e consentimento

Este formulário tem por objetivo validar, por especialistas, os itens do IDCGT-AP, desenvolvido para diagnosticar competências gerenciais e transversais na Administração Pública.

Escala de julgamento dos especialistas: 1 = inadequado/muito baixo; 2 = baixo; 3 = regular; 4 = adequado; 5 = muito adequado. Para consenso, consideram-se notas iguais ou superiores a 4.

Instruções de uso

1. Cada especialista deve atribuir notas de 1 a 5 para Clareza, Pertinência, Relevância e Representatividade.

2. O Coeficiente de Validade de Conteúdo (CVC) de cada item é calculado pela média das notas dividida pela nota máxima (5). O consenso considera o percentual de notas ≥ 4.

3. Decisão por item, conforme o CVC:

I. CVC ≥ 0,80 → Aprovado (item aprovado).

II. CVC entre 0,70 e 0,79 → Revisão (item sujeito a revisão).

III. CVC < 0,70 → Exclusão/Reformulação (item passível de exclusão ou reformulação).

4. Os resultados (Painel e Dados coletados) são de acesso exclusivo do administrador e não ficam visíveis ao especialista respondente. A cada envio, a resposta é encaminhada por e-mail ao administrador e armazenada no sistema para consulta posterior.

5. No Painel é possível analisar os resultados de forma cumulativa (todos os especialistas) ou individualmente por especialista, com o cálculo sendo atualizado a cada nova resposta.

6. Os dados podem ser exportados em CSV para análise em Excel, JASP, SPSS ou Power BI.
